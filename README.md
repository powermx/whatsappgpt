## Instalacion
**Instalar en RDP/Windows ✅**

```bash
$ git clone https://github.com/Sansekai/Wa-OpenAI
$ cd Wa-OpenAI
$ npm install
$ node index.js
```
**Instalar en Termux ✅**

```bash
$ pkg install git nodejs -y
$ git clone https://github.com/Sansekai/Wa-OpenAI
$ cd Wa-OpenAI
```

**Ejecutar**
```bash
$ node index.js
```
